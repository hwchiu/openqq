Meaning

這個結果代表什麼

Kata runtime 本身沒壞

`/dev/kvm` 與 guest kernel 都已驗證，表示 VM 邊界這條線不是空談。

OpenShell control plane 也沒壞

Gateway、agent-sandbox controller 都能正常運作，阻塞點不是 control plane。

真正卡住的是 sandbox 啟動模型

只要這條路還要求 privileged bootstrap，Kata 目前就不是實際可用的 OpenShell 路線。
