這個 repo 現在是專業的 runtime security lab 文件站。核心入口是四套架構比較技術參考（全新專業風格，支援 Light/Dark，高對比工程師閱讀）。你可以直接選獨立路線：`k3s + gVisor`、`k3s + OpenShell + runc`、`k3s + OpenShell + gVisor`、`k3s + KubeArmor + runc`。

這個站現在分兩部分

  • Install Paths：告訴你怎麼搭起不同組合
  • Matrix：告訴你四套環境用哪些共用測試被比較
  • Lab Pages：告訴你每條組合實際驗證出了什麼

目前最穩主線

2026-06-08 live rerun 後，k3s + OpenShell + runc 仍是功能最完整的落地路線。`OpenShell + gVisor` 被確認為 `DEGRADED`，`KubeArmor + runc` 目前在 SA token block 案例上是 `FAIL`。

Open comparison matrix

目前已經有標準 test catalog、matrix runner、發佈用 JSON，以及 GitHub Pages 的比較頁。未來只要重跑 `install-comparison-matrix.sh` 與 `run-comparison-matrix-tests.sh`，網站就能更新成新的四環境比較結果。
