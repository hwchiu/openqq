四套環境
直接裝起來

這一頁只回答部署問題。Repo 現在同時提供四個獨立 Terraform root，加上四支對應的 installation script。最新基線已統一為 `K3s 1.31 + CRI-O 1.31`，而且已完成 `2026-06-09` 的四環境 live rerun。

一鍵入口

./scripts/install-comparison-matrix.sh

每套環境各自擁有獨立 state、resource group 與 kubeconfig。

Install Paths

四條獨立安裝線

Path 01

K3s + gVisor

只驗證 runsc 與 RuntimeClass，不混 OpenShell。

./scripts/install-k3s-gvisor.sh
Path 02

K3s + OpenShell + runc

目前最穩的完整主線，含 OpenShell 核心能力。

./scripts/install-k3s-openshell-runc.sh
Path 03

K3s + OpenShell + gVisor

獨立驗證 OpenShell 在 runsc 路徑下保留與退化的能力。

./scripts/install-k3s-openshell-gvisor.sh
Path 04

K3s + KubeArmor + runc

和 OpenShell 無關的 runtime security 對照線。

./scripts/install-k3s-kubearmor-runc.sh
Matrix

Comparison Matrix

如果你的目標不是單條路線，而是同時保留四套獨立環境做比較，直接使用 matrix 入口。清理時也用對應的 destroy 腳本回收全部 Azure 資源。

./scripts/install-comparison-matrix.sh
./scripts/destroy-comparison-matrix.sh
